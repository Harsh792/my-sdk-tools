# My SDK

A Unity SDK providing runtime components and editor tools.

## Requirements
- Unity 6000.3 or newer

## Installation

### Option 1: Install from Disk
1. Open Unity
2. Go to **Window → Package Manager**
3. Click **+ → Add package from disk**
4. Select the `package.json` file

### Option 2: Install from Git
https://github.com/yourname/com.mycompany.mysdk.git
